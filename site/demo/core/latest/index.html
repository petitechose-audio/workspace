This demo runs the Core simulator in your browser (WASM). For live communication, run oc-bridge locally and allow WebMIDI/WebSocket permissions.

Launch

Open the simulator page. It will try to connect to ws://localhost:8100.

Run oc-bridge locally

Example (headless, controller=WebSocket, core port):

oc-bridge --headless --controller ws --controller-port 8100

If you installed via this dev env:

ms bridge install
